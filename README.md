## Install

You'll need Docker and docker-compose.

```sh
git clone https://github.com/sophiabrandt/fastapi-crud
cd fastapi-crud
```

## Usage

```sh
docker-compose up -d --build
```

## Run tests

```sh
docker-compose exec web pytest .
```
